MicroVM Agentic Fan-out

A coordinator decomposes one question into distinct sub-questions, then hands a different sub-question to each Lambda MicroVM — every VM carries its own private, snapshot-hot chDB, so each answers a different facet in parallel with no shared backend. The coordinator then synthesizes the partial answers into one briefing. Edit the question to have the coordinator LLM plan a fresh decomposition live.
